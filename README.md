# 📌 Overview

This project is a **Java-based Railway Reservation System** inspired by the real-world **IRCTC** ticket booking platform.  
It allows users to:

- 🚆 Search available trains
- 🎫 Book train tickets
- 🔐 Generate unique PNR numbers
- 💺 Reserve seats
- ❌ Cancel bookings
- 🧾 Print aesthetic railway e-tickets

The project is fully built using **Object-Oriented Programming (OOP)** concepts and provides a clean & interactive console UI experience.

# ✨ Features

## 🎟️ Ticket Booking
- Book tickets for available trains
- Dynamic fare calculation
- Passenger details management

## 🔐 PNR Generation
- Auto-generated unique PNR
- Uses Java UUID for randomness

## ❌ Cancellation System
- Cancel booked tickets easily
- Updates booking status instantly

## 🧾 Aesthetic Ticket Printing
- Beautiful railway e-ticket UI

- Professional ticket layout

- Booking date & time generation
- Formatted timestamps
